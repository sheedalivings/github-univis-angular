Another very common task is to collect user input to perform actions based on the input. In this task, we will be creating input fields which convert amounts between different units. The following steps will guide you through one implementation of this.

  1. Add an input element below with an argument type="number" (to create a numerical input cell), a select element, and a couple of option elements inside the select. The options will be empty by default, so put some text inside each option element so the dropdown will show these.
  2. Add two-way binding to the input and select elements using [(ngModel)]. You'll need to create variables (we will refer to them as valueOneNumber and valueOneUnit) in the .ts file to bind to the values of these inputs. You can check this is correct by using interpolation to display the variables you have created, using .
  3. The variable lengthUnitsKeys is an array of strings, each being a different unit. Use an *ngFor directive to replace the current option elements with the given units.
  4. We now have a functioning input pair, for a number and a unit! Create a second pair of inputs, including separate variables, but still using lengthUnitsKeys as the list of units.
All that remains is to set up the interaction between the input pairs. We will do this by defining functions to convert the first value to the second and vice versa. These will be called when either values' number or unit fields are changed. We will call the variables valueTwoNumber and valueTwoUnit for consistency.
  1. Define a function which sets valueTwoNumber to the value of
    valueOneNumber * lengthUnits[valueOneUnit] / lengthUnits[valueTwoUnit]. Then bind this function to the (change) event in the first input and select elements. The second number should now update when the first number or unit change.
  2. Repeat the previous step, but updating the first number based on the second number and units.
