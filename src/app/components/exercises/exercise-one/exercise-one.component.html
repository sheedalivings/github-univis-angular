Exercise 1.4.

Now let's try to use the DataTableComponent, like the other tables in this app.

  1. Find the selector for DataTableComponent and insert an instance of the component below. Also note that there are three variables marked as inputs using the @Input decorator, being data, properties and rowIndexing.
  2. The component requires data to be bound to its [data] property. In exercise-one-solution.component.ts, inside ExerciseOneComponent, define a property as an array of some data e.g.
    someSongs = [
      { title: 'If I\'m Being Honest', duration: 279, album: 'Human' },
      { title: 'She', duration: 245, album: 'Human' },
      { title: 'Guiltless', duration: 208, album: 'Guiltless' },
      { title: 'Four Tequilas Down', duration: 142, album: 'Build a Problem' },
    ]
    and bind it to [data].
  3. The component also requires a list of strings in [properties] to know what columns to display e.g.
    columns = ['title', 'duration', 'album']
  4. The component also comes with an optional parameter [rowIndexing] which can be set to true to show an index column.
